# Bowling ruby game

### How to use

1. Open your terminal.

2. Check if you have ruby installed with `ruby -v`.

2.1 If not installed use for example `rvm install 2.3.0` or google more ways to install it

3. Choose whether you want to clone or download the repository

3.1 Download:
... Click on the Clone or download button
... Click on Download ZIP
... Choose where to save the repository
... Extract the downloaded folder

3.2 Clone:
... Check if you have git installed with `git --version`
... If not installed use for example `sudo apt-get install git` or google more ways to install it
... Click on the Clone or download button
... Copy the link given
... Go to the terminal and cd into the folder you will clone the app
... Use `git clone git@github.com:juanbedon/bowlingGame.git`

3.3 Run the app
... cd into the bowlingGame folder
... `ruby bowling.rb`
... Follow in-game instructions


### About

1. Made with ruby 2.5.0p0

2. Version 1.0 (Feb 2019)

3. Repository contains 4 files:
... **bowling.rb** (logic of the game)
... **bowlingSpec.rb** (TDD file with RSpec)
... **guide.md** (how I approached the app process)
... **README.md** (instructions)
